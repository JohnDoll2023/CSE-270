Print, Pr

Print object
(Print object1...)

Prints out one or more objects. If an object is a list, the outermost brackets are not printed. Writes a new line afterwards.

Print "abc

abc

print [the end of the line]

the end of the line

(print "abc "def "ghi)

abc def ghi

See also Show, GraphicsType, Type.

Show

Show object

Print object, then start a new line. if object is a list, the outermost brackets are printed.

Show [the end of the line]

[the end of the line]

(Show "abc "def "ghi)

abc def ghi

See also Print, GraphicsType, Type.

Type

Type object
(Type object1 object2...)

Print an object or objects - do not start a new line. Remove outer brackets for a list.

Type [the end of the line]

the end of the line

See also Print, GraphicsType, Show.
